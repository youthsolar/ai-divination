## ⚠️ 已知問題與風險

### 🚨 **阻塞性問題**

1. **ECPay 測試環境限制**
   - 訂單號重複導致無法重複測試
   - 必須升級到生產環境

2. **CheckMacValue 計算敏感性**
   - 編碼問題可能導致簽章失敗
   - 已修正大部分情況，需持續監控

### ⚠️ **潛在風險**

1. **Vercel 服務穩定性**
   - 外部依賴可能影響服務可用性
   - 建議準備備援方案

2. **Zoho Creator API 限制**
   - Public Key 認證方式的安全性
   - API 調用頻率限制

3. **多對話並行開發**
   - 功能整合可能出現衝突
   - 需要版本控制協調
