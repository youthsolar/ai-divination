## 未來動態定價方案

#### 價格結構：
```
- 基礎符令：200元（一般問題、日常煩惱）
- 進階符令：360元（工作發展、學業考試）
- 專業符令：500元（感情問題、財運規劃）
- 頂級符令：800元（人生重大決策、健康問題）
```

- **基礎（200元）**：綜合、其他

- **專業（500元）**：愛情、財運、投資
- **頂級（800元）**：健康、重大決策、家庭

### 🧮 **方案 C：混合定價模型**

#### 計算公式：
```
最終價格 = 基礎價格 + 問題複雜度係數 + 五行稀有度係數 + 個人化程度係數

基礎價格：200元
問題複雜度係數：0-200元
五行稀有度係數：0-100元
個人化程度係數：0-150元
```

## 實現架構

### 📊 **資料表結構**

#### `Talisman_Pricing` 表單設計：
```
欄位設計：
- Question_Type (單選): 愛情、工作、財運、健康、學業、人際、綜合、其他
- Element_Type (單選): 金、木、水、火、土
- Base_Price (數字): 基礎價格
- Complexity_Multiplier (小數): 複雜度係數 (1.0-2.5)
- Seasonal_Adjustment (小數): 季節性調整 (0.8-1.2)
- Is_Active (是/否): 是否啟用此定價規則
- Effective_Date (日期): 生效日期
- Notes (長文字): 定價說明
```

#### 範例資料：
```
愛情 + 水 = 基礎價格300 + 複雜度係數1.5 + 季節調整1.0 = 450元
工作 + 金 = 基礎價格250 + 複雜度係數1.2 + 季節調整1.0 = 300元
健康 + 木 = 基礎價格350 + 複雜度係數2.0 + 季節調整1.0 = 700元
```
