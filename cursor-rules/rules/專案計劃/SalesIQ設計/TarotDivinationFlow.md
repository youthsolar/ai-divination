# SalesIQ 塔羅占卜對話卡片流程設計

## 🎯 設計目標
建立用戶友善的分步驟資料收集流程，提升占卜體驗品質並減少輸入錯誤。

## 📝 對話卡片設計（7 步驟）

### 卡片 1：歡迎與姓氏收集
```
標題：✨ 歡迎來到 AI 塔羅占卜 ✨
內容：親愛的朋友，歡迎體驗我們的專業塔羅占卜服務！

為了為您提供最精準的占卜解讀，請先告訴我您的姓氏：
```
- **輸入類型**：文字輸入
- **欄位名稱**：`lastName`
- **提示文字**：請輸入您的姓氏（例如：王）
- **驗證**：必填，1-5 個字符

### 卡片 2：名字收集
```
標題：🌟 很高興認識您！
內容：謝謝您，{lastName} 先生/女士！

接下來請告訴我您的名字：
```
- **輸入類型**：文字輸入
- **欄位名稱**：`firstName`
- **提示文字**：請輸入您的名字（例如：小明）
- **驗證**：必填，1-10 個字符

### 卡片 3：聯絡電話
```
標題：📱 聯絡資訊
內容：{lastName}{firstName}，為了在占卜完成後能與您聯繫，請提供您的手機號碼：
```
- **輸入類型**：電話號碼
- **欄位名稱**：`phoneNumber`
- **提示文字**：請輸入手機號碼（例如：0912345678）
- **驗證**：09 開頭，10 位數字

### 卡片 4：電子郵件
```
標題：📧 電子郵件
內容：占卜報告將透過電子郵件發送給您，請提供您的 Email：
```
- **輸入類型**：電子郵件
- **欄位名稱**：`email`
- **提示文字**：請輸入電子郵件（例如：example@gmail.com）
- **驗證**：Email 格式驗證

### 卡片 5：出生日期
```
標題：🎂 出生資訊
內容：為了更精確的占卜解讀，請提供您的出生日期：
```
- **輸入類型**：日期選擇器
- **欄位名稱**：`dateOfBirth`
- **提示文字**：請選擇您的出生日期
- **驗證**：1920-2010 年範圍

### 卡片 6：出生時間（可選）
```
標題：⏰ 出生時間
內容：如果您知道出生時間，可以提供更詳細的命理資訊：
```
- **輸入類型**：時間選擇器
- **欄位名稱**：`timeOfBirth`
- **提示文字**：請選擇出生時間（不確定可選擇 12:00）
- **驗證**：可選，24小時格式

### 卡片 7：占卜問題
```
標題：🔮 您的問題
內容：最後，請詳細描述您想要占卜的問題。越具體越能得到精準的指引：
```
- **輸入類型**：多行文字
- **欄位名稱**：`userQuestion`
- **提示文字**：請詳細描述您的問題（例如：我想知道我的工作發展如何？）
- **驗證**：必填，10-200 字符

```
標題：✅ 資訊確認
內容：請確認您的資訊：

👤 姓名：{lastName}{firstName}
📱 電話：{phoneNumber}
📧 郵件：{email}
🎂 生日：{dateOfBirth}
⏰ 時間：{timeOfBirth}
🔮 問題：{userQuestion}

確認無誤請點擊「開始占卜」，需要修改請點擊「重新填寫」。
```

## 🔄 用戶體驗流程

### 正常流程
1. 歡迎 → 2. 收集姓氏 → 3. 收集名字 → 4. 電話 → 5. 郵件 → 6. 生日 → 7. 時間 → 8. 問題 → 9. 確認 → 10. 開始占卜

### 錯誤處理
- **格式錯誤**：即時提示重新輸入
- **網路中斷**：資料自動保存
- **中途退出**：下次進入恢復進度

### 個性化引導
- 使用用戶姓名稱呼
- 根據問題類型給予特殊提示
- 貼心的鼓勵話語

## 💡 技術實作要點

### Session 資料結構
```json
{
  "step": 3,
  "lastName": "王",
  "firstName": "小明",
  "phoneNumber": "0912345678",
  "email": "example@gmail.com",
  "dateOfBirth": "1990-01-01",
  "timeOfBirth": "14:30",
  "userQuestion": "我想知道我的工作發展如何？",
  "completed": false
}
```

### 驗證規則
- **姓氏/名字**：中文、英文、1-10字符
- **電話**：09開頭，10位數字
- **郵件**：標準 Email 格式
- **日期**：合理範圍內的日期
- **問題**：10-200字符，具體描述

### 錯誤恢復
- 每步完成後立即保存
- 意外中斷可恢復到最後完成步驟
- 提供「重新開始」選項
