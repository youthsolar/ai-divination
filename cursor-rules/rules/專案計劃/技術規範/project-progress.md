# AI占卜系統開發進度表

本文件用於追蹤「AI靈學占卜系統」的開發進度、功能規格，並記錄重要決策。

---

## 核心戰略方向 (2025/09/11 更新)

我們的核心戰略是**集中資源，優先打通一條能最快實現營收的自動化路徑**。

**核心產品價值主張：** `本命 (Destiny) + 占卜 (Divination) + 解法 (Solution) = 人人都需要的生命之書`

**核心商業路徑：**
**客戶 -> SalesIQ 提問 -> AI自動占卜 -> AI推薦商品/符令 -> (兩階段確認) -> ECPay付款 -> 自動交付**

**雙重營收模式：**
1. **事件型收費**：單次占卜 NT$360（提供3000字完整報告+符令，7~14天效期）
2. **年費會員制**：NT$3,600~6,000/年（每月流月運勢、每季塔羅占卜、符令9折、年度專屬報告等）

為此，專案優先級已進行重大調整，將 `ECPay` 金流的穩定性驗證與 `Zoho SalesIQ` 的全自動化整合置於所有任務的核心。

### 🎯 技術開發準則 (2025/09/11 新增)

**Zoho One 生態系優先策略：**
- 身為 Zoho One 生態系擁有者及最高管理員，除 GCP、n8n、Twilio 等必要外部服務外，我們盡可能在 Zoho 生態系內解決問題
- 優先使用 Zoho Creator、CRM、SalesIQ、Analytics、Campaigns、Social 等原生整合
- 確保資料統一性、權限一致性、維運簡化與成本最優

### 💡 核心行銷策略：超倒楣運勢示警系統 (2025/09/11 新增)

**目標：** 提高免費占卜訪客的購買轉換率

**策略概念：** 針對僅做免費占卜的訪客建立「超倒楣通知」自動化行銷系統
- **技術實現：** 透過 Zoho Creator 運算統計八字相沖月份，識別高風險時期
- **自動化通路：** 整合 Zoho Campaigns 自動發送個人化運勢預警
- **行銷主軸：** 「過了這週，如果一樣超倒楣趕快購買這個符令」
- **預期效果：** 利用「損失厭惡心理」與「時效性」提升符令購買率
