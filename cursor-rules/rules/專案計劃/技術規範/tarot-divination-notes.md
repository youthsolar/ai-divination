# 本命塔羅占卜法筆記

## 牌陣說明

本命塔羅主要使用三張牌，輔以一張可選的牌。

-   **第一張（中間）：本世功課**
    -   代表個人在此生需要學習和面對的主要課題。
-   **第二張（左側）：時事地**
    -   反映當前的生活狀況、環境以及遇到的事件。
-   **第三張（右側）：業力因果**
    -   揭示過去的行為、選擇所帶來的因果關係與業力影響。
-   **第四張（可選）：有無佛緣**
    -   用於探測個人與修行或佛法的緣分。
    -   **注意** ⚠️：當第三張牌顯示的業力過於沉重時，不建議抽第四張牌。

## 抽牌方式

-   **方法**：數字抽牌法，結合「陰（逆位）」與「陽（正位）」卦。
-   **原理**：此方法與《易經》的六十四卦相對應，透過數字來選牌，並以牌的正逆位來區分陰陽。
-   **時間**：由於是占算「本命」相關議題，此抽牌法與占卜當下的時間點沒有直接關聯。
