## 2. 用戶痛點與情緒切入

### 2.1 最高頻關鍵詞（371篇合計）

| 詞彙 | 出現次數 | 意義 |
|------|----------|------|
| 挽回 | 2,300+ | 核心需求，壓倒性主題 |
| 復合 | 606 | 明確目標 |
| 分手 | 500 | 觸發事件 |
| 成功 | 249 | 必要的結果承諾 |
| 希望 | 88 | 情緒出口 |
| 無助 | 43 | 入場情緒 |
| 感謝 | 31 | 信任轉化 |

### 2.2 用戶入場情境（從文章歸納）

1. **突然被分手**：毫無預警、不知道原因、完全接受不了
2. **對方有新歡**：閨密、前任、網友、小三出現
3. **找過其他老師失敗**：浪費錢、等待沒效果、被說「業障太重」
4. **自己嘗試挽回失敗**：傳訊息被已讀、LINE 被封鎖
5. **婚姻危機**：婆媳問題、外遇、冷戰、夫妻關係冰凍

### 2.3 情緒弧線（典型故事結構）

```
痛苦入場 → 自己嘗試失敗 → 找其他老師失敗 
→ 遇到「真正的老師」→ 老師「感應到對方的心」
→ 配合施法 → 等待期間有微小進展 → 成功復合
→ 感謝老師
```

**關鍵轉折點**：「老師感應到了我不知道的事」——這是建立信任的核心場景。

## 3. 話術與文案框架

### 3.1 主力話術模式

**觀心居（know-heart）**
- 主打「感情挽回術」具體術語，技術感強
- 強調「李老師」個人通靈能力：「老師感應到對方心裡真正要的是什麼」
- 文章末固定 CTA：LINE ID + 微信，免責聲明「挽回成效因人而異」
- 人物設定精確（職業、外貌、故事細節）→ 讓用戶產生代入感

**賽思心靈（sunnylove）**
- 主打法術品牌：「和合術」、「斬桃花」、「鎖心術」
- 文章更長、情緒更完整，用戶自述視角（第一人稱）
- 強調老師「通靈照片感應」：「只給了照片，老師就說出很多準確的事」
- 服務標籤化（方括號分類）：【感情挽回】【和合術】【斬桃花】

### 3.2 高效情緒觸發句型

```
「我實在無能為力，只能試試...」
「老師說出許多我從來不知道的事，讓我震驚」
「終於感覺找到了真正能幫助我的人」
「挽回成功那天，我哭了很久」
「如果不是老師，我不知道現在會怎樣」
「只給了照片，老師就感應到了一切」
```

### 3.3 故事角色設定規律

- 主角：高學歷/有職位（店長、設計師、醫師、會計師、老板）→ 強化「連這種人都需要幫助」
- 對方：有缺點但可被喚回（外遇、冷淡、工作狂）
- 障礙：前任、第三者、家庭壓力

## 4. 服務與定價模式

### 4.1 服務模式

兩家都採用「**先諮詢、後施法**」模式：
1. 用戶聯絡（LINE/WeChat/表單）
2. 初步諮詢（了解情況 + 建立信任）
3. 老師「感應/占卜」後提出方案
4. 收費施法

**定價完全不透明**——不在網站上標示，一切透過私訊決定。
這是刻意的：讓用戶先建立情感連結後，才進入價格討論（降低前期抗拒）。

### 4.2 內容行銷策略

- 大量真實感案例（第一人稱 or 第三人稱敘述）
- SEO 關鍵詞密度極高：「挽回男友」「和合術」「斬桃花」在每篇反覆出現
- 免責聲明但不突出（藏在結尾）
- 無明確成功率數字，但案例量本身就是社會證明

## 5. 你的差異化優勢

### 5.1 他們沒有、你有的

| 你有 | 競品沒有 |
|------|---------|
| **系統化占卜流程**（塔羅 + 易經雙重）| 依賴老師個人感應，無可複製性 |
| **即時 LINE 回應**（AI 生成，幾分鐘內）| 等待老師手動回覆，時間不確定 |
| **透明入口**（生日填入就開始）| 必須先聯絡→溝通→等老師 |
| **符令具象化商品**（NT$360 實體/數位）| 服務費用不透明 |
| **冷靜機制**（7天冷卻）| 可能被反覆消費情緒 |
| **個人命理側寫**（Clients_Report）| 每次都要重頭說故事 |

### 5.2 你的弱點（需補強）

- 目前缺乏「老師感應」的神秘感和信任背書
- 案例故事量零（競品靠幾百篇建立信任）→ **需要開始建立用戶回饋文章庫**
- 「AI生成」認知可能降低靈性信任感 → 對外說法要謹慎

## 6. Rowy 專區：SimplyBook 服務定位建議

### 6.1 競品服務 vs 你的老師服務

| 競品（老師個人）| 你的老師（SB預約）|
|----------------|-----------------|
| 聯絡→等待→不知道費用 | 線上預約→透明時段→明確收費 |
| 無法事先評估 | 可看老師簡介/專長 |
| 完全依賴個人信任 | 品牌背書 + 個人特色 |
| LINE 私訊制 | 系統化預約管理 |

### 6.2 定位建議

- **進階諮詢**定位：AI 占卜是入口（低門檻），老師是深度服務（高價值）
- 用「LIFF 占卜結果頁」推薦對應老師 → 自然引流到 SimplyBook 預約
- 強調老師由 Jeffery 親自培育 → 品質保證，不是隨機配對
