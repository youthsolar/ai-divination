# Copy Pack v1（MVP 上線文案包）

Last updated: 2026-02-18

> 目標：用一致口吻「安撫 + 給方向 + 不刺激焦慮 + 有行動」，並能支援冷卻與回購。

## 1) 首次入口（引導輸入）
**歡迎詞**
- 「我在。先深呼吸一下，告訴我兩件事就好：①國曆生日 ②你想問的問題。」

**格式提示**
- 「生日格式：YYYY/MM/DD（例：1992/03/18）」
- 「問題盡量一句話說清楚，例如：『他還會回來嗎？』、『這份工作要不要換？』」

## 2) 免費結果（固定結構模板）

**段落 1｜安撫**
- 「我知道你現在很急、也很想立刻得到肯定的答案。先不用逼自己馬上做決定。」

**段落 2｜結果（主結論一句話）**
- 「這題的方向是：__（例：先穩住自己、再談行動）__。」

**段落 3｜3 點建議（可落地）**
- 「(1) __」
- 「(2) __」
- 「(3) __」

**段落 4｜冷靜機制說明（建立公信力）**
- 「為了避免情緒上來一直追問、反而把你推得更焦慮，我們同一題會有冷卻時間。建議你隔幾天再回來看會更準。」

**段落 5｜CTA（不逼單）**
- 「如果你希望把這個方向『落成一個可執行的護持』，可以購買一張符令（NT$360）。它會在接下來一段時間，幫你把能量穩住在正確的路上。」

## 3) 冷卻期回覆（不打模型的固定回覆）
- 「我知道你還在想同一件事，但這題目前在冷卻期，先不要再用占卜把自己推得更緊。」
- 「你可以在 **{cooldown_until}** 之後再回來（我會在那時候用更新的狀態幫你看）。」
- 「這段時間請你只做一個小行動：__（例：把想問的問題改成可行動版本/先睡一覺/去散步10分鐘）__。」

## 4) 付款成功 & 交付

**付款成功（立即）**
- 「收到了，你的符令已啟動。接下來我把內容交付給你（請保留 7 天）。」

**交付（Token URL 版）**
- 「這是你的符令連結（7 天內有效）：{token_url}」
- 「提醒：盡量在安靜的時候看完，照上面的『一個行動』去做，效果會最好。」

**交付（圖片版）**
- 「我把你的符令直接送到這裡了（建議你先存到相簿）：」
- 「（圖片）」

## 5) 回購提示（第 2–5 張）
- 「你這題的護持是有進度的：目前 **{step}/5**。」
- 「下一次可以回來的時間：**{next_purchase_earliest_at}**。」
- 「你不需要每天來問；你只需要按節奏走，我會陪你把路走穩。」

## 6) 上限結束（5/5）
- 「這題的符令進度已經完成（5/5）。接下來更關鍵的是『你要怎麼做』。」
- 「如果你願意，我可以給你兩條路：
  1) 我幫你把問題改寫成更可行動的版本，再做一次新的占卜
  2) 你想要真人老師介入，我給你預約入口（下一階段服務）」
